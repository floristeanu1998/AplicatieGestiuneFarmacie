# Aplicație de gestiune a unei farmacii

Acest proiect reprezintă o aplicație consolă realizată în C# pentru gestionarea stocului și a operațiunilor de bază dintr-o farmacie.

## Operații implementate:
* **Adăugare medicament**: Permite introducerea unui nou medicament în sistem (nume, preț, stoc etc.).
* **Editare/ștergere medicament**: Permite modificarea detaliilor unui medicament existent sau eliminarea acestuia din stoc.
* **Afișare listă medicamente**: Afișează toate medicamentele disponibile în mod curent în farmacie.
* **Căutare medicament**: Permite găsirea rapidă a unui medicament după nume sau alte criterii.
